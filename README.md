# Mood_Tracking

## Description
Application web permettant d’enregistrer et suivre son humeur quotidienne avec possibilité d’ajouter une note personnelle.  
Ce projet est le **vingt-septième** du défi personnel **100 projets en 2026**.

---

## Objectifs du projet
- Enregistrer une donnée associée à une date
- Gérer un état quotidien unique
- Implémenter un CRUD simple
- Visualiser un historique personnel
- Structurer un mini dashboard émotionnel

---

## Plateforme
- Web (navigateur)

---

## Technologies utilisées
- HTML
- CSS
- JavaScript (Vanilla)
- LocalStorage
- Chart.js

---

## Fonctionnalités
- Sélection d’une humeur (😊 😐 😢 😡 😴 etc.)
- Ajout d’une note facultative
- Enregistrement automatique avec la date du jour
- Affichage de l’historique des humeurs
- Modification ou suppression d’une entrée
- Sauvegarde locale des données

---

## Design & UX
- Sélection d’émotions sous forme d’icônes arrondies
- Couleurs douces et rassurantes
- Historique clair et organisé
- Interface minimaliste et intuitive
- Responsive (mobile et desktop)
